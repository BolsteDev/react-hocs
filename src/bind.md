## default(hocs) ⇒
Use a flat structure to bind many connect statements to a component
instead of nesting them in a deep callback structure

**Kind**: global function  
**Returns**: function – A function that takes a component and returns a Component  

| Param | Type | Description |
| --- | --- | --- |
| hocs | <code>function</code> | any number of higher order functions that take a component and return a component |
